## Installation

```bash
npm install @salad-ui/spacing
```

## Usage

### Margin

```jsx inline=false
import styled from 'styled-components';
import {marginTop, marginRight} from '@salad-ui/spacing';

const Feature = styled.div`
  ${marginTop(2)}
  ${marginRight({mobile: 0, desktop: 3})}
  background-color: gold;
`;

render(<Feature>The most amazing feature!</Feature>);
```

- `margin` or `m` - applies margin on all sides
- `marginX` or `mx` - applies margin on the left and right
- `marginY` or `my` - applies margin on the top and bottom
- `marginTop` or `mt` - applies margin on the top
- `marginRight` or `mr` - applies margin on the right
- `marginBottom` or `mb` - applies margin on the bottom
- `marginLeft` or `ml` - applies margin on the left

### Padding

```jsx inline=false
import styled from 'styled-components';
import {paddingY, paddingX} from '@salad-ui/spacing';

const Content = styled.div`
  ${paddingY(2)}
  ${paddingX({mobile: 0, desktop: 3})}
  background-color: gold;
`;

render(<Content>Some very serious copy.</Content>);
```

- `padding` or `p` - applies padding on all sides
- `paddingX` or `px` - applies padding on the left and right
- `paddingY` or `py` - applies padding on the top and bottom
- `paddingTop` or `pt` - applies padding on the top
- `paddingRight` or `pr` - applies padding on the right
- `paddingBottom` or `pb` - applies padding on the bottom
- `paddingLeft` or `pl` - applies padding on the left

### Using the `css` prop

There are often times when you will need to apply a one-off spacing value to a component. Rather than create a whole
new styled component, you can use\* [`styled-component`'s `css` prop](https://medium.com/styled-components/announcing-native-support-for-the-css-prop-in-styled-components-245ca5252feb).

```jsx previewable=false
import {Heading} from '@salad-ui/typography';
import {marginBottom} from '@salad-ui/spacing';

<Heading size="large" css={marginBottom(3)}>
  Design System
</Heading>;
```

\* The `css` prop requires the `babel-styled-components` plugin.
